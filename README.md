# Teddy's Portfolio

A personal portfolio site built with Laravel and Blade, showcasing my projects, skills, and experience as a software engineer focused on backend development, low-level systems, and firmware. The site pulls live data from the GitHub API and uses an AWS Lambda endpoint to generate AI-powered insights about each technology and project on the fly.

🔗 **Live site:** [adeyneouskpoto.com](https://www.adeyneouskpoto.com)

---

## ✨ Features

- **Live GitHub integration** — repository stats, language breakdown, and project cards pulled in real time from the GitHub REST API
- **Animated skill rings** — interactive SVG circles with count-up animations for Frontend, Backend, Machine Learning, Data Analytics, and Mobile Development
- **AI-powered insights** — click any technology or project card to get a generated summary via AWS Lambda + API Gateway
- **Custom donut chart** — vanilla JavaScript canvas chart showing my most-used languages
- **Responsive design** — works smoothly across desktop, tablet, and mobile
- **Built from scratch** — no off-the-shelf templates; every component styled by hand
